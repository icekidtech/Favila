# Favila Cleaning - Website

A modern, responsive website for Favila Cleaning Services. This project showcases our professional cleaning services, company values, and commitment to quality.

## Project Overview

Favila Cleaning is a professional cleaning service provider dedicated to maintaining clean, healthy, and welcoming spaces for our clients. This website serves as a digital storefront to present our services, team, and why clients should choose us.

## Features

- **Responsive Design** - Mobile-friendly layout that works on all devices
- **Service Showcase** - Detailed presentation of our cleaning services
- **Company Information** - Learn about our vision, mission, and values
- **Why Choose Us** - Highlight our competitive advantages and expertise
- **Modern UI** - Built with Tailwind CSS for a professional appearance
- **Fast Performance** - Optimized with Vite for excellent load times

## Technology Stack

- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS + PostCSS
- **Code Quality**: ESLint
- **Package Manager**: pnpm

## Getting Started

### Prerequisites

- Node.js (14+)
- pnpm

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   pnpm install
   ```

### Development

Start the development server:
```bash
pnpm dev
```

The application will be available at `http://localhost:5173`

### Build

Create a production build:
```bash
pnpm build
```

### Preview

Preview the production build locally:
```bash
pnpm preview
```
